How it works

Four steps from video to TikTok

Download

Paste a YouTube URL. The app downloads the video automatically.

Transcribe

Whisper AI transcribes the audio and generates timed subtitle segments.

Cut & Edit

ffmpeg crops to 9:16, adds subtitles, branding, and a progress bar.

Publish

Clips are queued and published to TikTok via the Content Posting API.

TikTok API Integration

Transparent API usage

Products used

FilvockCutsApp integrates with TikTok's Login Kit and Content Posting API to authenticate users and publish video content on their behalf. No data is collected or shared beyond what is required for publishing.

Scopes requested

The following permissions are requested during TikTok authentication:

user.info.basic video.upload video.publish
